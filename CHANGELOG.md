Initial public release. Limenex is a deterministic stateful governance layer for
AI agents — see the [README](./README.md) for features, quickstart, and examples.

### Added

- `python -m limenex validate [path]` CLI command for loading and validating a
  policies YAML file, with clear error messages and skill/policy counts.
